## Codespell Dictionary

The autocorrect dictionary (`src/dictionary/base_safe.json`) is derived from the [codespell](https://github.com/codespell-project/codespell) project's dictionary files.

- **Primary File**: `dictionary.txt`

### Attribution

This project incorporates typo corrections from the codespell dictionary, which is based on English Wikipedia content. The dictionary has been filtered and processed for use in this autocorrect extension, but the original typo-correction mappings are derived from the codespell project.

### Modifications

The dictionary has been:
- Filtered for conservative corrections (excluding ambiguous words)
- Processed to prefer UK English spellings
- Converted to JSON format for use in this extension
- Enhanced with additional safe short typos
